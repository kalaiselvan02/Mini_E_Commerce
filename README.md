# Mini_E-Commerce

A Mini E-Commerce application built with Spring Boot, featuring JWT-based authentication and role-based access control.

#Features

Admin Role: Add products with image upload, manage inventory.

User Role: View products securely.

Authentication: Login using JWT tokens.

Security: Passwords stored with BCrypt, secure REST APIs with Spring Security.

Database: MySQL used to store users and products.

#Tech Stack

Java 21, Spring Boot 3

Spring Security, JWT, BCrypt

MySQL

REST APIs
